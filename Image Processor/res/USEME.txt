In order to run our program, you have to load in an image first, using the load button that is shown on the
top left. Once an image is loaded, you can choose any of the following image manipulation commands listed on
the left. In order to brighten or darken an image, you click the 'Brighten/Darken' button, and it will pop up
another section, where you will enter a positive or negative number, depending on whether you wnat to brighten
or darken an image, respectfully. In order to greyscale an image, you click the 'Greyscale' button, and it will
pop up another section, where you will choose one of the red, green, or blue components to greyscale. In order
to save an image, you click the 'Save an Image' button, where you can choose the file path of where you want
the image to be saved. For all other image manipulations buttons, you just simply press on the button of what
 manipulation you want.

------------------------------------------------------------------------------------------------------------

In order to run a script using this program, you must run the main method within the 'ImageUtil' class and
write the commands inside the console.

The script commands that are support by our program are:

load image-path image-name
save image-path image-name

flip image-name horizontal dest-image-name
flip image-name vertical dest-image-name

brighten image-name increment dest-image-name
darken image-name increment dest-image-name

greyscale image-name red dest-image-name
greyscale image-name green dest-image-name
greyscale image-name blue dest-image-name
greyscale image-name value dest-image-name
greyscale image-name intensity dest-image-name

filter image-name sharpen dest-image-name
filter image-name blur dest-image-name

transform image-name sepia dest-image-name
transform image-name luma dest-image-name

---------------------------------------------------------------------------
